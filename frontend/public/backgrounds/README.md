# Background images (drop your photos here)

Add high-quality, application-related photos here and the app will use them as section backgrounds.

Recommended:
- Format: `.jpg` (or `.webp`)
- Size: 1920×1080 (or larger), landscape
- Compression: ~200–600 KB each

## Expected filenames

- `temple-interior.jpg` — temple interiors / darshan
- `yamuna-aarti.jpg` — Yamuna aarti / ghats at night
- `parikrama.jpg` — parikrama / lanes / devotees walking
- `hotel-room.jpg` — hotel room / premium stay shot

## Where they are used

- Home “Featured Stays / Popular Rooms” uses `hotel-room.jpg`
- Home “Popular Tour Packages” uses `parikrama.jpg`
- Home “Why Choose Us” uses `temple-interior.jpg`

If a file is missing, the app still builds, but that section will just show the pattern background.
